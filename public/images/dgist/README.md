# DGIST Internship Images

Place your images in this folder with the following names to have them automatically appear in the portfolio:

## Required Images

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `lab-setup.jpg` | Hero image - Lab setup or workstation | 1920x1080 |
| `noisy-trace.png` | Power consumption trace showing noise | 1200x600 |
| `pipeline.png` | System workflow/pipeline diagram | 1200x800 |
| `scatter-plot.png` | W&B optimization results scatter plot | 1200x800 |

## Gallery Images (Life in South Korea section)

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `team-photo.jpg` | Photo with PACL team | 800x800 (square) |
| `posco.jpg` | POSCO Steel Factory tour | 800x800 (square) |
| `busan.jpg` | Busan coastal city | 800x800 (square) |
| `temple.jpg` | Korean temple visit | 800x800 (square) |

## Tips for Best Results

1. **Optimize images** - Use tools like TinyPNG to compress images
2. **Use consistent aspect ratios** - Gallery images work best as squares
3. **High resolution** - Use at least 2x the display size for retina displays
4. **WebP format** - Consider using .webp for better compression (change the extensions in cv.js)

Once you add the images, uncomment the `<img>` tags in `/src/components/Experience.jsx` (search for "Uncomment when you have actual images").
